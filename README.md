# Scraty

## Project management tool based on Scrum

## Installation

Bootstrap project with Python >= 3.5:

```bash
python bootstrap.py
bin/buildout
```

To bootstrap the app simply run `npm` in this scraty folder:

```bash
bin/npm install
```

## Usage

The app is served on defaul-port `8080` by running:

```bash
bin/app
```
